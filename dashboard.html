📚 How to Use PenCraft Pro

Get the most out of the AI engine and WordPress tools.

✍

The Dual-Pass Rewriter

Standard AI bypassers use simple synonym swapping. PenCraft Pro's "Super Rewrite" mode does two things:

1. Plagiarism Pass: Inverts clauses, rearrises paragraphs, and replaces verbs to guarantee 0% match.
2. Humanize Pass: Injects "burstiness" (short + long sentences) and "perplexity" (unpredictable words) so it passes GPTZero and Originality.ai.

🔄

API Fallback System

You can add keys for Gemini, OpenAI, and Claude in Settings.

If you hit an API limit (e.g., Gemini says `429 Too Many Requests`), the app will automatically try OpenAI, and then Claude, ensuring your workflow is never interrupted.

💻

WordPress formatting

When you click "Clean Copy" or "Copy to WP", the app automatically strips out markdown asterisks `**` and hidden styling that usually breaks Elementor or Gutenberg. Paste fearlessly!
